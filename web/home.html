Tools

Self-service utilities for understanding and cleaning up your Azure estate. Pick a tool to get started — more will be added here over time.

🧹
Idle Resource Scanner
Scans every subscription and flags resource groups with no recent activity across Activity Log, metrics, and last-modified signals. Surfaces estimated monthly cost, Azure Advisor findings, orphaned resources, locks, and potential cleanup savings.
Cost & Cleanup Open →
🔐
Privileged Access Scanner
Scans subscriptions and management groups for standing (active) privileged role assignments held directly by users and service principals — access that bypasses group-based governance. Surfaces a per-scope heatmap, orphaned assignments, and PIM-eligibility recommendations.
Security & Access Open →
👤
Entra User Scanner
Scans Microsoft Entra ID for stale and orphaned user accounts — members who haven't signed in for months, expired passwords, and unused guest invitations. Distinguishes password-based members from B2B guests and recommends accounts to disable or remove.
Identity & Access Open →
🏷️
Tag Auditor
Recurses every subscription, resource group, and resource to audit tag usage against your required-tag rules. Flags objects missing required keys, with blank or invalid values, or inconsistent key casing — evaluating effective (inherited) tags.
Governance Open →
📊
Log Analytics Cost Projector
Pick a subscription, resource group, and Log Analytics workspace, then project cost per table from each table's billable ingestion and current retention settings — priced live from the Azure Retail Prices API. Forecast spend over 1, 3, 6, or 12 months.
Cost & Cleanup Open →
📈
Quota Usage Scanner
Scan every subscription for Compute, Network, and Storage quota usage across the regions each subscription actually uses. Surfaces the quotas closest to their limit, flags them critical or warning, and recommends an action for each.
Governance Open →
🔒
Storage Security Posture
Audits every storage account for public blob access, HTTPS-only transfer, minimum TLS, default-allow network ACLs, and shared-key access — ranked by exposure.
Security & AccessOpen →
🌐
Attack Surface
Inventories internet-facing resources — assigned public IPs and exposed storage, SQL, Key Vault, and Cosmos endpoints — and ranks the tenant's external exposure.
Security & AccessOpen →
🛡️
NSG Risk Map
Maps network security groups and flags risky inbound rules — any-to-any, broad source ranges, and management ports (RDP/SSH) open to the internet.
Security & AccessOpen →
📜
Certificate Expiry Monitor
Surfaces App Service certificates that are expired or expiring soon, with days-to-expiry and a renew/review recommendation per item.
Security & AccessOpen →
🛟
Defender Secure Score
Reads the Microsoft Defender for Cloud Secure Score per subscription — overall posture plus the per-control breakdown, ranked by how far each control falls short.
Security & AccessOpen →
🔑
App & SP Credential Expiry
Surfaces app-registration and service-principal client secrets and certificates that are expired or nearing expiry across the tenant.
Security & AccessOpen →
